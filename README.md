### Baixar dependências
```bash
pip install -r requirements.txt
```

### Rodando o script
```bash
python3 main.py
```

<br />
### Utilizando os serviços
#### Comandos:
<ul>
  <li>get-tasks: lista as tasks</li>
  <li>post: criar nova task</li>
  <li>update: atualizar task</li>
  <li>delete: deletar task</li>
</ul>

```bash
./script <comando> --help
```
